# SpellingCorrect_GUI

#### 介绍
自然语言处理（NLP）- 一个英文拼写纠错系统；
功能：给出拼写错误的单词，返回一个正确的单词，或者返回一个与输入单词最接近的单词；
IDE：pycharm；python3.5；PYQT做界面；
数据：
使用big.txt作为语料库。
